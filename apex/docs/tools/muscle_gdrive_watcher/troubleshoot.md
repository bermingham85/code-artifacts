# TROUBLESHOOT - muscle_gdrive_watcher

| Symptom | Likely Cause | Fix |
|---|---|---|
| Old files reappear | State file lost or changed | Restore the prior state file or accept a controlled re-scan |
| No files detected | Folder ID wrong or service account lacks access | Verify folder sharing and ID |
| State file corrupt | Interrupted write or manual edit | Move corrupt file aside and restart with documented re-scan |
