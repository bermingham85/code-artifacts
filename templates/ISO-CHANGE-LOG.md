## Format

Each entry records one approved change.
Draft iterations do not get entries — only approved transitions.

```
### [Version] | [Date] | [Author]
**Status change:** [FROM] → [TO]
**GitHub PR:** [link]
**GitHub SHA:** [first 8 chars]
**Reason:** [Why was this change made?]
**What changed:** [What specifically was added, removed or modified?]
**Impact:** [Does anything else need to change as a result?]
```
